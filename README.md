# Mossbank

A small static field guide to the bryophytes: what mosses are, common temperate
species, how to grow them, and the many uses they have been put to.

Plain HTML and CSS, no build step. Open `index.html` locally or deploy the
folder to any static host.

## Structure

- `index.html` — the single page
- `style.css` — styles
- `assets/` — photographs (Wikimedia Commons, licences credited in the footer)
